# 교안 봇 제작 (V7.0.0) 기능 설명서

## 1. 개요 및 주요 특징
이 애플리케이션은 게임 기획 전문가 양성 과정을 위한 **'AI 기반 교안 자동 생성 에이전트'**입니다. Gemini API를 활용하여 교육 과정 및 차시별 목차를 기획하고, 세부 교안, 퀴즈, 다이어그램, 관련 이미지를 자동으로 렌더링 및 생성합니다. 

## 2. 주요 기능 및 UI 구성

### 2.1. 과정 및 차시 관리 플랫폼
- **과정 개요 (LNB & 다이어그램)**: 
  - 좌측 편의 LNB(목차 패널)와 중앙의 무한 캔버스 다이어그램 뷰포트를 통해 전체 교과와 차시별 진행 상황을 시각적으로 파악할 수 있습니다.
  - 마우스 드래그 앤 드롭으로 교과의 순서를 변경하고 미니맵을 통한 뷰포트 이동이 가능합니다.
- **차시 상세 편집기 (Editor)**: 
  - 개별 차시를 클릭하면 우측 편집기가 나타납니다. `교안 생성` 버튼을 눌러 AI 기반으로 초안부터 다단계 이어쓰기 자동화 과정을 거치며 대형 마크다운 교안을 생성합니다.
  - 생성 후 퀴즈 문제 생성, 서식/문체 변환(교과서, 대화형, 실무형 등), 분량 조절 기능(길게/짧게)을 지원합니다.
  - 기존 마크다운(`.md`) 파일을 직접 업로드해 수정할 수도 있습니다.
- **메인 퀘스트 (Level Test)**: 
  - 해당 교과의 모든 일일 퀘스트(실습)를 종합하여 최종 평가(레벨테스트)를 구성하는 특수 모듈입니다.

### 2.2. 인텔리전트 이미지 웹 검색 및 생성 프록시
본문에 들어갈 이미지를 수집하고 확보할 때 다음 4단계 체인을 순차적으로 사용하여 API 할당량과 검색 품질을 최적화합니다.
1. **로컬 캐시 확인 (IndexedDB)**: 이전에 탐색한 이미지가 캐시에 있다면 즉시 렌더링.
2. **웹 검색 (Google Custom Search)**: 
   - 텍스트 문맥을 추출하여 가장 완벽한 영문 검색어를 AI가 도출한 뒤 구글에서 실제 이미지를 수집합니다. 
   - 수집된 이미지는 자동적으로 VLM(비전 언어 모델) 검열을 거쳐 썸네일 낚시나 부적절한 이미지 발생을 필터링합니다. (API 할당량 관리를 위한 다중 Key 롤링 지원)
3. **스톡 검색 (Pixabay)**: 웹 검색이 막혔을 경우, 2차 검색으로 무료 스톡 사진을 검색합니다.
4. **AI 이미지 생성 (Gemini DALL-E 기반)**: 모든 검색이 실패하거나 적합하지 않은 경우 최종적으로 AI가 이미지를 직접 렌더링(생성)합니다.

### 2.3. 데이터 무제한 보존 및 입출력 (Export/Import)
- **로컬 아카이브 (IndexedDB 기반 히스토리)**: 
  - 생성되거나 삭제되는 수정 내역은 대용량 브라우저 로컬 DB인 IndexedDB에 자동 세이브됩니다.
  - **캐시 보관함 / 히스토리 패널**: 저장된 데이터를 복구하거나 불필요한 데이터를 휴지통 기능처럼 손쉽게 삭제 및 일괄 PDF/MD 다운로드 할 수 있습니다.
- **로컬 PC 폴더 직접 연동**: 
  - 브라우저를 벗어나 작업자의 실제 내 PC(로컬 폴더)와 연동(`빈 폴더 연결`, `기존 폴더 연결`)하여 파일이 실시간으로 외부 PC 폴더에 `.json`, `.md` 물리적 파일로 안전 보관됩니다.
- **ZIP 및 PDF 내보내기**:
  - 교과 및 차시 개별/단위 모두 PDF 저장 또는 압축(ZIP) 파일 형태로 산출할 수 있습니다.

## 3. API 키 발급 및 설정 방법

우측 상단의 **설정(톱니바퀴) 아이콘**을 클릭하여 아래 세 가지(또는 필수 한 가지) API 키를 설정해야 정상 동작합니다.

### 3.1. Gemini API Key (필수 설정값)
- **용도**: 텍스트 생성 파이프라인(교안 작성, 퀴즈, 문체 변경, 번역), 이미지 검열관(VLM), 및 AI 이미지 생성.
- **발급 방법**:
  1. [Google AI Studio](https://aistudio.google.com/app/apikey)에 접속 후 구글 계정으로 로그인합니다.
  2. `Create API key` 버튼을 클릭해 새 프로젝트에 키를 생성하십시오.
  3. 영문+숫자로 구성된(예: `AIzaSy...`) API Key를 복사합니다.
  4. 본 애플리케이션 화면 우측 상단의 톱니바퀴 > **Gemini API Key** 란에 붙여넣기 후 `저장`을 클릭합니다.
- **사용 모델 지정**: `LLM 모델` (권장: `gemini-2.5-flash`), `이미지 생성 모델` (권장: `gemini-3.1-flash-image-preview`)

### 3.2. Google Custom Search 엔진 및 API 연동 (선택 권장)
- **용도**: Deep Research 기반 시각 자료 검색용 (다이어그램, 실제 게임 스크린샷 등)
- **발급 방법**:
  1. **Search Engine ID (CX)**:
     - [Google Custom Search](https://programmablesearchengine.google.com/controlpanel/all) 창에 들어가 검색엔진을 추가합니다. 대상은 '전체 웹' 혹은 레퍼런스가 될 커뮤니티로 지정.
     - 설정 화면에 나오는 **검색 엔진 ID**를 복사해 앱의 `Search Engine ID (CX)` 칸에 넣습니다.
  2. **API Keys**:
     - [Google Cloud Console](https://console.cloud.google.com/)에서 새 프로젝트 생성 뒤 **Custom Search API**를 활성화합니다.
     - **사용자 인증 정보** 탭에서 사용자 제한 없는 'API 키'를 생성하여 앱 **API Key(s)** 칸에 입력합니다. 일괄 검색 시 429(할당량 초과)가 뜰 경우, 다른 계정/키를 여러 개 발급 받아 콤마(`,`)로 구분해 다중 등록하면 자동으로 순회 롤링(Rotate)이 작동합니다.

### 3.3. Pixabay API Key 연동 (선택)
- **용도**: 구글 웹 검색 실패, 한도 초과 시 Fallback 대체재로 활용하는 라이선스 프리 스톡 사진 호출.
- **발급 방법**:
  - [Pixabay API 문서](https://pixabay.com/api/docs/)에 로그인한 뒤 하단의 본인 할당 `Key` (예: `1234567-abcdef...`)를 찾아 앱의 `Pixabay API Key` 란에 넣습니다.

## 4. 상세 이용 워크플로우 (사용 가이드)

1. **상태 연결 및 세팅**: 
   - 톱니바퀴 아이콘을 클릭하여 Gemini API Key 등 필수 정보를 세팅합니다.
   - 상단 메뉴의 **데이터 연동** 버튼을 눌러 내 문서함 하위에 `새 작업 폴더`를 만들고 연결하는 것을 권장합니다 (새로고침 시 증발 방지 및 영구 소장).
2. **교과 과정 뼈대 잡기**:
   - 좌우를 둘러보며 과정 개요에서 **`+ 새 교과 추가`** 버튼으로 대주제를 만들고 제목을 지어줍니다.
   - 교과 내에서 여러 **차시 모듈**(`+ 새 차시 추가`)들을 배치합니다. 드래그앤드롭으로 순서를 바꿀 수 있습니다.
3. **AI 교안 세부 편찬**:
   - 작성할 해당 차시를 클릭한 후, 편집 패널에서 **`교안 생성`**을 누릅니다.
   - 팝업에 "게임 루프, 최적화" 처럼 이번에 꼭 들어갈 **핵심 개념**들을 입력해주면 AI가 이를 고려하여 초안부터 디테일 구조화까지 이어쓰기로 상세 문서를 뽑아줍니다.
4. **리터칭 (이미지/퀴즈/어투 수정)**:
   - 교안 안에 [이미지 영역] 텍스트가 있다면, 이미지 상단 우측에 있는 **이미지 재생성**을 눌러 적절한 시각 자료를 삽입시킬 수 있습니다.
   - **퀴즈 생성**을 통해 모듈 마무리 평가를 덧붙이거나 상단의 **서식 변환** 툴바를 눌러 전체 교안의 분위기를 캐주얼하거나 엄격하게 변경합니다.
5. **저장 및 다운로드**:
   - 작업 완료 후엔 페이지를 새로고침하거나 창을 꺼도 이력이 자동 저장(DB 및 폴더)되니 안심합니다.
   - 최종본 제출이 필요할 경우 상단의 **`PDF로 내보내기`** 혹은 **`데이터 연동 -> zip으로 내보내기`**를 통해 파일 포맷팅합니다.
